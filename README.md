# SpatioBiomarker


**Spatial Transcriptomics NMF Analysis**
MATLAB code for Non-negative Matrix Factorization (NMF)-based analysis of Visium spatial transcriptomics data. This pipeline decomposes spatial gene expression data into cell type-specific spatial maps and expression profiles and quantifies their association with reactivity gene distributions.

-------------------------------------------------
Requirements

MATLAB R2019b or later
Seurat (R) — for SCTransform normalization of Visium data
No additional MATLAB toolboxes required beyond base MATLAB
